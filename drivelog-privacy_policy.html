차량 운행 일지

개인정보처리방침 · Privacy Policy

시행일: 2026년 1월 1일 기기 내 저장만 외부 서버 미사용
"차량 운행 일지" 앱(이하 "앱")은 사용자의 개인정보를 소중히 여기며, 개인정보 보호법 및 관련 법령에 따라 아래와 같이 개인정보처리방침을 운영합니다.
1수집하는 정보

앱은 서비스 제공을 위해 다음 정보를 수집합니다:

  • 위치 정보 (GPS): 운행 출발지/도착지 좌표 및 이동 경로 기록
  • 블루투스 기기 정보: 차량 연결 자동 감지를 위한 등록 기기 이름
  • 이메일 주소: 운행 기록 Excel 파일 발송을 위해 사용자가 직접 입력
  • 알림 설정: 리마인더 시간 등 앱 내 설정값
2수집 목적
  • 차량 운행 기록 생성 및 관리
  • 운행 경로·거리 자동 계산
  • 차량 블루투스/OBD2 연결 자동 감지
  • 운행 기록 이메일 발송
  • 앱 사용 환경 설정 유지
3정보의 보관 및 처리
  • 모든 데이터는 사용자의 기기 내부에만 저장됩니다.
  • 외부 서버, 클라우드 또는 제3자에게 전송·저장되지 않습니다.
  • 앱을 삭제하면 저장된 모든 데이터가 함께 삭제됩니다.
  • 백업 파일은 사용자가 직접 선택한 기기 내 폴더에 저장됩니다.
4제3자 제공

앱은 수집한 개인정보를 제3자에게 제공하지 않습니다. 단, 사용자가 직접 이메일 발송 기능을 사용할 경우 기기에 설치된 메일 앱 또는 사용자가 지정한 SMTP 서버를 통해 수신자에게 파일이 전달됩니다.

⚠️ SMTP 직접 발송 시 안내 사용자가 SMTP 서버를 직접 설정하여 이메일을 발송하는 경우, 해당 이메일 내용 및 첨부 파일(운행일지 엑셀)이 사용자가 지정한 외부 SMTP 서버로 전송됩니다. 이 경우의 데이터 처리는 해당 서버 운영자의 정책을 따릅니다.
5위치 정보 이용
  • 포그라운드 위치: 운행 시작/종료 시 현재 위치 확인
  • 백그라운드 위치: 운행 중 경로 실시간 추적 및 거리 계산
  • 위치 정보는 기기 내부에만 저장되며 외부로 전송되지 않습니다.
📍 백그라운드 위치 사용 목적 화면이 꺼지거나 앱이 최소화된 상태에서도 운행 중 거리·경로를 정확히 기록하기 위해 백그라운드 위치 권한이 사용됩니다. 수집된 위치 데이터는 기기 내부에만 저장되며 외부로 전송되지 않습니다.
6블루투스 정보 이용
  • 사용자가 직접 등록한 차량 블루투스 기기 이름만 저장합니다.
  • 연결 감지 목적 외에는 사용되지 않습니다.
  • 블루투스 MAC 주소 등 민감한 정보는 수집하지 않습니다.
7사용자 권리
  • 설정 > 데이터 백업/복원 메뉴에서 데이터를 내보내거나 삭제할 수 있습니다.
  • 설정 > 운행 기록 전체 초기화로 모든 운행 기록을 삭제할 수 있습니다.
  • 앱 삭제 시 기기에 저장된 모든 정보가 완전히 제거됩니다.
8아동 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

9방침 변경 안내

개인정보처리방침이 변경될 경우 앱 업데이트를 통해 변경 내용을 안내합니다.
